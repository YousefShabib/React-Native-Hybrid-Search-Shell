# React Native Hybrid Search Shell

**GitHub:** [React-Native-Hybrid-Search-Shell](https://github.com/YousefShabib/React-Native-Hybrid-Search-Shell)

## Overview
A Hybrid Shell app in React Native that wraps Google Search in a WebView.

## Features
- Full-screen WebView
- SafeAreaView support
- Works on iOS & Android

## Installation
```bash
git clone https://github.com/YousefShabib/React-Native-Hybrid-Search-Shell.git
cd React-Native-Hybrid-Search-Shell
npm install
npx expo start
